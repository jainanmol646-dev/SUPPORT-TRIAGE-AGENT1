# Support Triage Agent

## Overview
An intelligent terminal-based support triage agent that processes tickets, classifies user intent, detects high-risk cases like fraud or account compromise, retrieves relevant documentation, and decides whether to respond or escalate.

## Features
- Issue classification
- Risk detection (fraud, hacked account)
- Document-based response
- Escalation handling

## How to Run
1. Install Python
2. Install dependencies:
   python -m pip install pandas
3. Run:
   python main.py

## Tech Stack
- Python
- Pandas
